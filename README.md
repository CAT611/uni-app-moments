# uni-app-moments
基于uni-app的仿微信朋友圈模版

#### 功能简介

* 点赞
* 评论+回复评论
* 图片预览
* 发布页
* 图片上传

#### 使用方法

1. 直接将uni-app-moments拖到HbuilderX中运行即可
